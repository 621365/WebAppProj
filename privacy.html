Privacy Policy

This site uses no cookies and does not collect any user data except when the user explicitly requests editors to collect data through the Article Submission page functionality. All other data is processed on the user-end (or front-end) and is not sent back. This website has no back-end server and relies on a JavaScript file containing all article information as it is the only database of information collected.
